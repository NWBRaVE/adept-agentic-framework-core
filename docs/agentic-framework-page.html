Get Started in Three Steps

It's easy to get the framework running on your own machine with Docker. For a detailed guide on adding new tools, refer to the full tutorial in the repository.

1

Clone the Repository

Get the source code on your local machine.

git clone https://tanuki.pnnl.gov/cloudhub/agentic-framework.git
2

Configure Environment

Copy the example .env file and add your LLM API keys.

cp .env.example .env
3

Build and Run

Use Docker Compose to build the images and launch the containers.

docker compose up --build -d
