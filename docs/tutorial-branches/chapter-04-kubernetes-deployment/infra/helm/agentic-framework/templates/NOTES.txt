Your Agentic Framework has been deployed.

1. Get the application URL by running these commands:
{{- if .Values.ingress.enabled }}
  {{- range .Values.ingress.hosts }}
  http{{ if $.Values.ingress.tls }}s{{ end }}://{{ .host }}{{ range .paths }}{{ .path }}{{ end }}
  {{- end }}
{{- else if contains "LoadBalancer" .Values.streamlit_app.service.type }}
  export SERVICE_IP=$(kubectl get svc --namespace {{ .Release.Namespace }} {{ include "agentic-framework.fullname" . }}-streamlit-app -o jsonpath='{.status.loadBalancer.ingress[0].ip}')
  echo "http://${SERVICE_IP}:{{ .Values.streamlit_app.service.port }}"
{{- else if contains "NodePort" .Values.streamlit_app.service.type }}
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "agentic-framework.fullname" . }}-streamlit-app)
  export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
  echo "http://$NODE_IP:$NODE_PORT"
{{- else }}  echo "To access the Streamlit UI, run this command and visit the URL below:"
  echo "Visit http://127.0.0.1:8501 to use your application"
  kubectl port-forward --namespace {{ .Release.Namespace }} svc/{{ include "agentic-framework.fullname" . }}-streamlit-app 8501:{{ .Values.streamlit_app.service.port }}
{{- end }}

The application is composed of multiple microservices. You can check the status of all pods by running:
kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}
